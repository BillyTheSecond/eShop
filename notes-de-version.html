Version 4.2 :

Refonte de la page d'accueil qui est maintenant presque entièrement automatisée. La fonction HomePageSuggestion() permet l'affichage intelligent des différents appareils classés par catégories.
Cette fonction fonctionne de la manière suivante :

  • Recherche de toutes les catégories disponibles
  • Recherche de tous les produits d'une catégorie
  • Classe ces produits par date décroissant de sortie
  • Ecris le code HTML contenant le titre de la section et, éventuellement, une icone d'illustration
  • Ecris les boites des différents produits (il y a maximum 6 produits présentés par catégories sur la page principale. Ce sont donc les 6 plus récents.)
  • Répète ces étapes pour toutes les catégories détéctées

Version 1.0 :

Ajout d'une barre de navigation bleu pastel en haut de la page. Elle contient le nom de la page web et un bouton en haut à droite.

Le menu de navigation comporte le nom de l'utilisateur s'il est "connecté". Dans le cas contraire, il lui propose de se "connecter". Il fait donc appel à plusieurs fonctions JavaScript.

Lorsque l'utilisateur rentre son pseudo (dans une boite de dialogue qui s'affiche au clic de "se connecter"), son pseudo s'enregistre sur son appareil dans un cookie. Ainsi, le site le reconnaîtra toujours s'il actualise la page, ouvre à nouveau l'onglet ou éteint son appareil.

Une fois connecté, l'utilisateur peut cliquer sur son nom pour afficher deux options : "Se déconnecter", qui supprime tous les cookies et actualise le contenu de la page, et "Mon Profil", qui renvoie vers la page profil.html .

La page Profil permet à l'utilisateur de connaître et de modifier les données qui ont été enregistrées sur son ordinateur et que le site utilise. Un bouton lui propose également de renseigner plus d'informations.

Sur la page Profil, le bouton comportant le nom de l'utilisateur (en haut à droite) renvoie vers la page principale (index.html)
